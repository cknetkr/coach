Concrete Example
문제 상황: log₂(x-1)=3 같은 로그방정식을 보면 바로 계산하지 말고 진수 조건 x-1>0부터 씁니다.
학생이 고민해야 할 포인트: "왜 x=9만 쓰면 안 되고, x>1을 먼저 써야 하지?"를 이해해야 합니다.
서술형 안전 답안 흐름: 조건 확인 → 로그의 정의 적용 → x-1=2³ → x=9 → 조건 만족 확인 → ∴ x=9.

📌 자주 나오는 오류 유형 요약
유형 A — 로그 조건 누락
$\log_a M$에서 반드시 $a>0$, $a\neq1$, $M>0$ 3가지 모두 답안에 명시해야 부분 점수를 받습니다.
유형 B — 거듭제곱근 분기 실수
n이 짝수일 때 $\sqrt[n]{a^n} = |a|$ (절댓값), 홀수일 때 $\sqrt[n]{a^n} = a$ — 이를 혼동하면 부호 오류 발생.
유형 C — 로그 부등식 방향
밑이 $0 < a < 1$이면 부등호 방향이 뒤집힙니다. $\log_{1/2} x > \log_{1/2} 3 \Rightarrow x < 3$
